# دليل الشاشات والأزرار — Tiandy Store

> دليل بصري/وظيفي بالعربي لكل شاشة في التطبيق: **شو وظيفتها، مين بيشوفها، وكل زر شو بيعمل.**
> الهدف: تفهم الواجهة بسرعة بدون ما تنبش بالكود.
>
> 🧠 **قاعدة العمل ("اسأل حالك"):** قبل أي تعديل على شاشة، افتح قسمها هون، افهمها زر-زر، واسأل حالك: شو الهدف؟ مين بيشوف شو؟ وين الأزرار بتتغيّر حسب الدور؟ — وبعد التعديل حدّث هالملف.

## كيف نقرأ هالدليل

كل شاشة موثّقة بنفس القالب:
- **المسار** (الـ URL) و**مين بيوصلها**.
- **الهدف** بسطر واحد.
- **مين بيشوف شو** (زائر / تاجر معتمد / أدمن / سوبر أدمن).
- **الأزرار والعناصر** — جدول: العنصر ← شو بيعمل.

الأدوار:
- **زائر** — مش مسجّل دخول. بيشوف الكتالوج بلا أسعار.
- **تاجر/مقاول معتمد** (`dealer`, approved) — أسعار + سلة + طلب.
- **أدمن / سوبر أدمن** — لوحة الإدارة كاملة (السوبر أدمن كمان بيدير المدراء ويشوف التكلفة/الربح).

## شاشة الكتالوج — `/products`

**المصدر:** `src/app/products/page.tsx` · **مين بيوصلها:** الكل (زائر ومسجّل).

**الهدف:** تصفّح المنتجات، الفلترة والبحث، وفتح صفحة كل منتج. الأسعار للتجار المعتمدين فقط.

### مين بيشوف شو
- **زائر:** شبكة المنتجات + صورهم وأسمائهم، **بلا أسعار** (مكتوب "מחיר לסוחרים")، مع شريط تنبيه أعلى الصفحة "الأسعار والطلب للتجار المعتمدين فقط · כניסה". ما في فرز.
- **تاجر معتمد:** أسعار + شارة "במלאי"/"אזל" + قائمة الفرز.
- **سوبر أدمن:** بالإضافة، سطر صغير بكل بطاقة فيه **سعر البيع / التكلفة / الربح والهامش%** (مخفي عن الباقين تلقائياً من قاعدة البيانات).

### الأزرار والعناصر

| العنصر | شو بيعمل |
|--------|----------|
| حقل **"חיפוש לפי שם או מק״ט"** + زر **חפש** | يبحث بالاسم العبري **أو** رقم الصنف (SKU) |
| زر **📷 סריקה** | يفتح ماسح الباركود/QR بالكاميرا — لمّا يمسح كود: يفتح المنتج مباشرة إذا تطابق، وإلا يبحث فيه |
| زر **קטגוריות ▾** (موبايل فقط) | يفتح/يغلق درج التصنيفات |
| قائمة **מיון** (فرز) | افتراضي / حسب الاسم / السعر تصاعدي / السعر تنازلي — *تظهر للي بيشوف الأسعار فقط* |
| مربع **"זמין במלאי בלבד"** | يعرض المنتجات المتوفرة بالمخزون فقط |
| الشريط الجانبي: **כל הקטגוריות** + اسم كل تصنيف | يفلتر الشبكة حسب التصنيف المختار |
| **بطاقة منتج** (ProductCard) | الضغط عليها يفتح صفحة المنتج `/product?slug=…` |
| شارة **אזל** على البطاقة | تظهر إذا نفد المخزون (للمسجّلين) |
| عدّاد **"N מוצרים"** فوق الشبكة | عدد النتائج الظاهرة بعد الفلترة |

> 🔎 **ملاحظات سلوك:** الكتالوج يعرض المنتجات المتاحة للبيع فقط (`is_orderable = true`) — الأدمن بيخفي منتج بإلغاء "למכירה" من `/admin/products`. الباركود بيتطابق بالتدرّج: باركود ← SKU ← slug.

## شاشات أخرى (للتوثيق لاحقاً)

> - السلة `/cart` والدفع `/checkout`
> - الحساب الشخصي `/account` (+ orders, quotes, security, profile)
> - لوحة الإدارة `/admin` (Operations Center + الإجراءات السريعة)
> - الإدارة: products, dealers, dealers/new, quotes, customer-prices, inventory, suppliers, purchase-orders, admins, settings
