## 1) باختصار شديد (الخلاصة بنص دقيقة)

أنت موزّع رسمي لكاميرات **Tiandy** في إسرائيل، وعندك **بوابة طلبات B2B إلكترونية ذكية** (تطبيق ويب قابل للتثبيت على الموبايل، دخول بالبصمة، لوحة عمليات بالذكاء التشغيلي).

- **منافسوك نوعان:**
  1. **منافسة العلامة** (Tiandy ضد Hikvision / Dahua / Provision / Uniview) — على مستوى المنتج نفسه.
  2. **منافسة الموزّع/المتجر** (أنت ضد موزّعي كاميرات المراقبة في إسرائيل: ZBM, HVI, Secure Group, Oded Electronics, Dahua Israel) — على مستوى تجربة الشراء والخدمة.

- **ميزتك الكبرى:** أغلب المنافسين في إسرائيل بيبيعوا عبر **كتالوج عرض بسيط + مكالمة/واتساب لتقديم الطلب**. أنت عندك **بوابة طلب فعلية بأسعار خاصة لكل زبون + ذكاء تشغيلي (ديون، مخزون، عروض، تنبؤ)** — هاد نادر جداً عند المنافس المحلي.

- **نقطة ضعفك الرئيسية:** **حجم/تنوّع الكتالوج وقوة العلامة.** Hikvision/Dahua أكبر و"أشهر" بالسوق؛ لازم تبيع قصة Tiandy (NDAA، Starlight، سعر/قيمة) بوضوح.

## 3) شو عندك أنت؟ (جرد ميزات متجرك من فحص الكود)

### للموزّع/الزبون
- كتالوج كامل + بحث (اسم/SKU) + فلترة فئات + "متوفر فقط" + منتجات مميّزة.
- **أسعار خاصة لكل زبون** (سعر مختلف للتاجر مقابل المقاول مقابل اتفاق فردي).
- سلة + **دفع من خطوتين** (شحن → طريقة دفع) + رقم طلب (TND-100001).
- **حدّ ائتمان + تنبيه رصيد مستحق** قبل الطلب + حد أدنى للطلب.
- سجل الطلبات + تتبّع الحالة + الفواتير.
- **عروض الأسعار:** الزبون يشوف العرض ويقبله → يتحوّل لطلب مباشرة.
- **إعادة طلب ذكية:** يقترح المنتجات اللي بيشتريها عادةً ويذكّره لما "يحين موعدها".
- **نبض الإنفاق:** يشوف صرفه الشهري واتجاهه.

### الدخول والأمان (نقطة تفوّق واضحة)
- **دخول بالبصمة / Face ID (Passkey)** على **عدة أجهزة** لكل حساب.
- دخول بيوزرنيم أو إيميل + كلمة مرور.
- أدوار: زائر / موزّع / مقاول / أدمن / سوبر أدمن، مع موافقة على الحسابات الجديدة.
- **تطبيق قابل للتثبيت (PWA)** على iPhone وأندرويد يفتح بملء الشاشة.

### للإدارة (الأقوى — قلب التميّز)
- إدارة منتجات/فئات/مخزون/موردين/أوامر شراء.
- **معالج إضافة تاجر** (4 خطوات) + إرسال بيانات الدخول **بالواتساب** تلقائياً.
- **تسعير لكل زبون** + حارس هامش (تنبيه "تحت التكلفة"/"هامش ضعيف").
- **لوحة عمليات حيّة:** إيراد، ربح، COGS، طلبات، مقارنة بالفترة السابقة، رسم بياني، أفضل المنتجات بالهامش، تصدير CSV، تحديث كل 60ث.
- **بريفينغ يومي:** قائمة مهام صباحية (طلبات تنتظر، ديون متأخرة، مخزون ناقص، عروض راكدة...).
- **ملف زبون 360°:** درجة خطر + تقادم الديون + نبض الإنفاق + منتجاته المتكررة + أزرار واتساب حسب حالته.
- **اكتشاف الزبائن النائمين** (توقّفوا عن الطلب) + **المخزون الميت** (مال مجمّد) + **تنبؤ نفاد المخزون** + اقتراح أمر شراء ذكي.
- **توصية سعر ذكية** عند عمل عرض سعر.
- **قوالب واتساب** جاهزة (تذكير دفع، عرض، فاتورة، استرجاع زبون، أمر شراء).

### إشعارات وتقنية
- **إشعارات Push** للأدمن عند كل طلب جديد.
- واجهة عبرية RTL كاملة، خلفية Supabase آمنة (RLS)، باركود/QR سكانر مدمج.

## 5) شو بميّزك فعلاً؟ (نقاط البيع الفريدة — استعملها بالتسويق)

1. **بوابة طلب ذاتية 24/7 بأسعار خاصة لكل زبون** — الزبون يطلب وحده بسعره المتّفق، بلا انتظار مكالمة. (معظم المنافس لسا "اتصل بنا").
2. **دخول بالبصمة على الموبايل** — تجربة بنكية حديثة، نادرة جداً عند موزّعي الكاميرات.
3. **ذكاء تشغيلي يخدمك أنت كصاحب عمل:** تعرف مين مدين، مين زبون "بدأ يغيب"، شو مخزون راكد، وإيمتى رح ينفد منتج — قرارات بالأرقام مش بالحدس.
4. **استرجاع زبائن واسترداد ديون بضغطة واتساب** بقوالب جاهزة.
5. **متوافق Tiandy NDAA** — ورقة رابحة للمشاريع الحكومية/الحساسة اللي Hik/Dahua ممنوعة فيها.
6. **تطبيق يُثبَّت على الهاتف** بشعارك، يفتح بملء الشاشة — حضور دائم بجيب الزبون.
